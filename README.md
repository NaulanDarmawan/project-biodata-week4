### ðŸ› ï¸ TUGAS HARI INI (STEP-BY-STEP)

1. **Clone Proyek**
   * Download folder ini ke laptop kalian dengan perintah:
   * `git clone https://github.com/NaulanDarmawan/project-biodata-week4.git`
   * Lalu masuk ke folder proyek: `cd project-biodata-week4`

2. **Buat Kamar Sendiri (Branch)**
   * Buat jalur kerja baru dengan nama panggilan kalian.
   * **Contoh:** `git checkout -b budi`

3. **Buat File Biodata**
   * Buka VS Code, cari folder **students**.
   * Buat file baru di dalam folder tersebut dengan format: **nama-lengkap.html**.
   * **Contoh:** `students/budi-setiawan.html`.
   * Isi file tersebut dengan kodingan HTML biodata kalian.

4. **Setor Hasil Kerja (Push)**
   * Simpan perubahan kalian ke GitHub:
   * `git add .`
   * `git commit -m "Tambah biodata Budi"`
   * `git push origin [nama-branch-kalian]`

5. **Lapor ke Tech Lead (Pull Request)**
   * Buka halaman GitHub, klik tombol **Compare & Pull Request**.
   * Tulis pesan: *"Izin gabung profil siswa dengan nama [Nama Kalian]!"* lalu klik **Create Pull Request**.
